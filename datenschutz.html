Der Schutz Ihrer personenbezogenen Daten ist uns wichtig. Diese Datenschutzerklärung informiert Sie darüber, welche Daten die App „Rentenrechner" erhebt, wie diese verwendet werden und welche Rechte Ihnen zustehen.

3. Erhobene Daten

Die App verarbeitet folgende Angaben, die Sie freiwillig eingeben:

  • Name (frei wählbar, zur Unterscheidung von Profilen)
  • Geburtsdatum
  • Anzahl der Beitragsjahre
  • Angabe einer Schwerbehinderung (GdB 50+)
  • Geschätzte monatliche Rente (optional)
  • Gewünschtes Renteneintritts­datum (optional)

4. Speicherung und Verarbeitung

Sämtliche Daten werden ausschließlich lokal auf Ihrem Endgerät gespeichert (mittels AsyncStorage). Es findet keine Übertragung an externe Server, Cloud-Dienste oder Dritte statt.

Die Daten werden ausschließlich zur Berechnung Ihres Renteneintrittsalters, der Abzüge und des Countdowns verwendet.

5. Keine Weitergabe an Dritte

Ihre Daten werden nicht an Dritte weitergegeben, verkauft oder anderweitig übermittelt. Die App enthält keine Werbung, keine Tracking-Dienste und keine Analyse-Tools.

6. Keine Internetverbindung erforderlich

Die App funktioniert vollständig offline. Eine Internetverbindung ist weder für die Nutzung noch für die Datenspeicherung erforderlich.

7. Löschung Ihrer Daten

Sie können Ihre Daten jederzeit löschen:

  • Einzelne Profile über die Benutzerverwaltung in der App
